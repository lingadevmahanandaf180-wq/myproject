# Simple DevOps Project

This project demonstrates a basic DevOps workflow using Linux, Git, GitHub, and Docker.

## Project Overview
- Created a simple static website
- Served the website using a server
- Containerized the application using Dockerfile
- Used Git and GitHub for version control

## Technologies Used
- Linux (Termux)
- Git & GitHub
- Docker
- HTML

## How to Run
1. Clone the repository
2. Build Docker image
3. Run the container
